plotting stereo sound on a plane

Lissa Juice is an audio visualization made by Noura Howell and myself inspired by Lissajous Curves. It is a stereo audio plot of two harmonizing oscillators on a 2 dimensional plane. Every dot drawn on the plane is a recent sample taken from the oscillators, the x dimension is a sample taken from the left oscillator the y component is a sample taken from the right oscillator. If the oscillators are harmonious, Lissa Juice creates a repeating pattern. When there is a detuned harmony between the oscillators and the sample rate the dots seem to travel around the screen.

Lissa Juice uses the Web Audio Api to play sound, so you will need to run it inside Chrome, or Chromium.
