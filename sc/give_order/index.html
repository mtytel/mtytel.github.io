Giving Order to Your Voice

Right now we have a lot of parameters to control and it still doesn't sound very exciting. A musician is only as good as his synth so first let's replace the awful default synth with this industrial sounding synth from the Berlin SuperCollider Users Group. There are a few parameters to the synth but one to notice is the 'index'. When 'index' is larger the synth will sound more percussive, when it is smaller the synth will sound less percussive or "softer" if you will. You can download their SynthDef pool if they still have it up here: http://www.fredrikolofsson.com/temp/enka012.zip 

(
  SynthDef(\krgn_gen_fmdevil,{
    arg out=0, freq=440, amp=0.2, index=3, detune=1.02, gate=1;
    var mod1, mod2, mod3, car, idx, env;
    env = EnvGen.ar(Env.adsr(0,0.4,0.3,0.2),gate: gate, levelScale: amp, doneAction: 2);
    idx = EnvGen.ar(Env.adsr(0,0.8,0.0,0.8),gate: gate, levelScale: index);
    mod1 = SinOsc.ar((freq / 12) * 2, LocalIn.ar(2), idx);
    mod2 = SinOsc.ar((freq / 12) * 6, mod1, idx);
    mod3 = SinOsc.ar([(freq / 12) * (9 * detune.neg), (freq / 12) * (9*detune)], mod2, idx);
    LocalOut.ar( mod3 * 0.25);
    car = SinOsc.ar(freq, mod3, env);
    Out.ar(out!2,car)
  }).store;
)

Now that we have a cool sounding synth, let's create some song structure. If we restrict our music to measures, then our random data creates a time signature and a rhythm or possibly a repeating motif. We can also add some variation by throwing in some random values to set the tempo, measure duration and base frequency. 

(
  var basefreq, notecomplexity, seed, measurelength, durations, speed;
  seed = 1000000.rand;
  thisThread.randSeed = seed;
  seed.postln;                    // Printing the seed so it's not lost forever.

  basefreq = 400.rand + 200;
  notecomplexity = 4.rand + 3;
  speed = 1.5.rand + 0.5;
  t = TempoClock();
  t.tempo_(speed);
  measurelength = 4.rand / 2 + 1;

  // Pconst will restrict this measure to |measurelength|.
  // Pseed will repeat the measure over and over... using our seed.
  p = Pseed(seed, Pbind(
    \numerator, Pwhite(1, notecomplexity),
    \denominator, Pwhite(1, notecomplexity),
    \dur, Pconst(measurelength, 1 / (2 ** Pwhite(1, 3))),
    \amp, Pwhite(0.1, 0.5),
    \legato, Pbrown(0.01, 0.4, 0.01),
    \instrument, \krgn_gen_fmdevil,  // Setting our synth
    \freq, Pkey(\numerator) / Pkey(\denominator) * basefreq,
  ));
  a = p.play(t);
)

(a.stop;)

This can come up with some groovy stuff or something really awkward so be sure to try it a few times.
