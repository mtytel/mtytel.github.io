Introduction to Algorithmic Composition

Algorithmic Composition is the technique of using algorithms to create music. An example of this is writing a computer program to come up with and play a song. There are a fair number of software packages out there that allow different levels of flexibility in writing these composing programs, from Data Flow software like PureData (where you drag and drop components), to more standard text based languages like SuperCollider.

Recommended knowledge

In this tutorial we will use SuperCollider, mostly using their Pattern library; James Harkins (a big name is SuperCollider land) has a great pattern tutorial A Practical Guide to Patterns if you're not well versed yet. As far as music knowledge goes, we will avoid the standard western scales. Instead, an understanding of harmonics is more useful in the coming chapters.
