FAQ!
What is Helm?
Helm is a software synthesizer. You use it to create electronic music on your computer.
Is Helm really free?
Yes, Helm is free (as in no money) but more importantly, Helm is free as in freedom! This means you control this software, it doesn't control you.
Don't you need to make money, to like.. eat?
Yes please donate, I'm very hungry.
I think I found a bug. What should I do?
Send me an email at matthewtytel@gmail.com and I'll fix it! Include information like what happened, a crash report if you have one, the operating system you're using (GNU/Linux, Mac OSX, Windows), if you're using the plugin version or standalone and any other info you think is useful.
I'd like to help out! What can I do?
Besides donating, testing and reporting bugs is very helpful. I'd also really love sound samples, songs and any patches you've created that I can distribute with Helm. I have some sweet stickers for sale too!
